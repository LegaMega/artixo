# ARTIXO

ARTIXO is a modern, minimalistic web-based art gallery.
It is designed as a quiet, museum-like environment for displaying digital artwork.

## Technologies
- **Front-end**: React, JavaScript (ES6+)
- **Styling**: CSS
- **Back-end**: Node.js
- **Build tool**: Vite
- **Database**: Supabase

## Getting started

### Prequisities

Ensure [Node.js](https://nodejs.org/en) is installed on your system.

### Installation

1. **Clone the repository**
    ```bash
    git clone https://github.com/LegaMega/artixo
    ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Start the development server**
   ```bash
   npm run dev
   ```
   Or using Vite directly:
   ```bash
   npx vite    # localhost only
   npx vite --host    # accessible on local network
   ```
   The application will be available at http://localhost:5173.
