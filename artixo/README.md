# ARTIXO

A modern, minimalistic online art gallery built for the web.
ARTIXO provides a seamless experience for browsing and displaying my artwork.

## Technologies
• **Front-end**: React.js, JavaScript (ES6+)
• **Styling**: CSS
• **Back-end**: Node.js
• **Build tool**: Vite
